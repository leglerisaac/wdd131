Overview

Purpose

The purpose of the Workout Tracker is to help users log their workouts easily and effectively by providing a user-friendly interface and useful tracking features. This is done by providing an intuitive interface for workout logging and a repository of common workouts to make it easier to create new workouts.

Audience

The primary audience for the Workout Tracker includes weightlifters, bodybuilders, and powerlifters. Within this group of people, the niche of lifters that this website caters to is those who use a science-based and structured approach to training.

Dynamic elements

The Workout Tracker will include dynamic elements such as real-time workout logging, the ability to search and sort exercises based on their attributes, and an optional dark mode. Real-time workout tracking is handled on the home page, as it is the primary purpose of this website. On the second page, there will be a repository of exercises that can be sorted and filtered based on what various parameters. Dark mode will be applicable on all pages of the site.

Wireframes

Workout Tracker

The Workout Tracker wireframe shows the main interface where users can log their workouts. The layout includes a section for entering exercises, sets, reps, and weights, as well as a button to complete the workout once at least one exercise has been added. What is not shown is the workout report that will be shown once the workout is completed, which will display an overview of the workout and some statistics. Workout tracker pulls exercise information from this open source, "Unlicense" database of exercises: https://github.com/yuhonas/free-exercise-db

Exercise Finder

Exercise finder allows the user to search for and filter exercises using attributes of the exercises. Not shown in the wireframe is the actual content of the exercise cards. Each card will contain the name of the exercise, a brief description, and an image or icon representing the exercise. Exercise finder pulls exercise information from the same database as the workout tracker.
